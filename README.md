# 概要
Football Sampleは、Swiftで書かれたサンプルプロジェクトです。
このプロジェクトは、フットボールに関連したアプリケーションの開発を目指しています。
特に、VIPERアーキテクチャを使用し、CallBackとCombineなど違いを理解することを目的としています。
